# 计数器应用

一个使用 React Hooks 实现的最小化前端计数器应用。

## 功能特性

### 核心功能
1. **计数器显示**：页面中央显示当前计数器数值，初始值为 0
2. **增减功能**：提供 +1 和 -1 按钮，分别用于增加和减少计数
3. **Reset 功能**：提供独立的 Reset 按钮，点击后将计数器重置为 0，并清空历史记录

### 历史记录
1. **记录操作**：记录所有非零的历史操作（+1 或 -1）
2. **格式要求**：历史记录只显示操作类型，不记录 Reset 操作
3. **数量限制**：最多显示 5 条最近的操作记录，超过则移除最旧的记录（先进先出）

### 边缘情况处理
1. **下限锁定**：计数器数值绝不能低于 0，当数值为 0 时，-1 按钮自动禁用
2. **历史记录管理**：自动维护历史记录的数量，确保不超过 5 条

## 技术栈

- **React 18**：使用 React Hooks 实现组件状态管理
- **HTML5**：基础页面结构
- **CSS3**：样式设计，包括响应式布局和动画效果

## 运行方式

### 方法一：直接打开 HTML 文件
1. 下载或克隆项目
2. 用现代浏览器打开 `index.html` 文件
3. 应用将立即运行，无需额外配置

### 方法二：使用 HTTP 服务器（推荐）
1. 安装 Python 3（如果尚未安装）
2. 在项目根目录运行以下命令：
   ```bash
   python3 -m http.server 10315
   ```
3. 在浏览器中访问 `http://localhost:10315/index.html`

## 使用说明

1. **增加计数**：点击 "+1" 按钮，计数器数值增加 1
2. **减少计数**：点击 "-1" 按钮，计数器数值减少 1（当数值为 0 时按钮禁用）
3. **重置计数器**：点击 "Reset" 按钮，计数器数值重置为 0，并清空历史记录
4. **查看历史记录**：在计数器下方的历史记录列表中，查看最近的 5 条操作记录

## 性能优化

- 使用 React Hooks 的 `useState` 进行状态管理，确保状态更新的高效性
- 历史记录数组的更新使用函数式更新，避免状态依赖问题
- CSS 动画使用 `transform` 和 `opacity` 属性，确保动画的流畅性

## 扩展建议

如果需要扩展这个应用，可以考虑以下功能：

1. **键盘支持**：使用键盘上的 "+"、"-" 和 "R" 键进行操作
2. **主题切换**：添加深色模式和浅色模式的切换功能
3. **数据持久化**：使用 localStorage 保存计数器数值和历史记录
4. **操作时间戳**：在历史记录中显示每一次操作的时间
5. **计数器上限**：设置计数器的最大值限制
6. **步数调整**：允许用户调整每次增减的步数（如 ±5、±10 等）
