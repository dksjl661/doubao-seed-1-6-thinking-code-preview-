# 会议室预订系统

一个基于 React 的简易会议室预订组件，满足以下核心需求：

## 核心功能

1. **时间段管理**：维护一天 8 小时（9:00 - 17:00）的时段状态
2. **时段选择**：支持点击选择一个或多个连续的未预订时段
3. **预订功能**：提供 "Book Now" 按钮，点击后将选中时段标记为已预订
4. **状态显示**：已预订时段用红色背景显示，不可再次点击

## 边缘情况处理

1. **连续性约束**：用户只能选择连续的时段，否则显示错误提示
2. **最大时长限制**：单次预订时长不能超过 3 小时，否则显示警告
3. **选择限制**：用户只能从第一个未预订的时段开始选择

## 使用方法

1. 启动开发服务器：
   ```
   npm start
   ```

2. 打开浏览器访问：
   ```
   http://localhost:3000
   ```

3. 选择可用时段（灰色），选中的时段会变成绿色
4. 点击 "Book Now" 按钮完成预订

## 组件说明

### MeetingRoomBooking 组件

主要功能组件，包含以下核心逻辑：

- **状态管理**：使用 React useState 管理时段状态和错误/警告信息
- **时段点击处理**：handleTimeSlotClick 函数处理时段选择逻辑
- **预订处理**：handleBookNow 函数处理预订逻辑和验证
- **样式管理**：内联样式管理不同状态时段的显示效果

### 时段状态

每个时段包含以下状态：

- `time`: 时间段（如 "9:00"）
- `isBooked`: 是否已预订（true/false）
- `isSelected`: 是否已选中（true/false）
